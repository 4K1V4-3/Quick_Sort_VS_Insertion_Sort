# Quick Sort VS Insertion Sort

## Description

Analyzation of the Quick Sort algorithm and its usefulness in practice./

This project includes two versions of the Quick Sort algorithm. In one version, just the arrays and subarrays are passed into the function. In the other, the arrays and subarrays are accompanied by indexes.
Also included is a breakdown of how the Quick Sort algorithm works.

## Analysis

Included in this project is a program that compares the speed of the Quick Sort algorithm to that of the Insertion Sort algorithm.
For an array of just 15,000 random integers, the following were the findings:

```shell
Insertion Sort =  7.874424236011691
Quick Sort =  6.134085101017263
```
